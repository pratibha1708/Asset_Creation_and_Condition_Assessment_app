Below are the various functionality that the app offers in brief.

User Ranking

This menu returns the users ranking in an alert message.

Add Layer-5 Closest Assets

This menu plots the five closest assets near the user's location.

Remove Layer- 5 closest assets

This menu removes the five closest assets plotted by add layer-5 closest assets menu.

Add Layer-5 last 5 reports, colour coded

This menu adds assets on the map that have last five condition report submitted by the user. The colour of the assets is given based on the condition of the assets.

Remove Layer-5 last 5 reports

This menu removes the layer: "add layer-5 last 5 reports, color coded.

Add Layer- not rated in last 3 days

This menu adds assets on the map which were not rated in last 3 days.

Remove Layer-not rated in last 3days

This menu removes the layer: "add layer-not rated in last 3 days
