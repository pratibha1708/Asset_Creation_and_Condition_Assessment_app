Open the App

Condition Form Pop ups
 -- The condition form appears as soon as the app is opened.
-- The user is asked to rate the asset which is nearest to them.

Fill form

Rate condition of asset
 -- Condition form provides asset information such as asset name and its installation date.
-- Rate the asset condition
-- Click on Save Asset Condition Button.

INFORMING USER WITH ALERT MESSAGES

1. COMPAIRING CONDITION
 -- When the user submits the form, alert message appears.
-- Alert message tells the user whether the condition of the information has changed or not.
-- Both new condition of the asset as given by user and asset's previous condiiton is displayed.
-- Click on OK to proceed further

2. REPORT SUBMITTED
 -- The user is notified that their form is succesfully submitted
-- Click on OK to proceed further

3. NUMBER OF REPORT SUBMITTED
 -- The user is notified the number of reports they have submitted till now.
-- Click on OK to proceed further

ASSET COLOR CHANGES

Asset Color changes based on the rating given by user
 -- The asset changes its colour based on the rating given the user and the app goes back to the home page.
-- Here, Green refers to best condition and dark red is the worst condition.
-- White means the asset is not rated yet and the condition is unknown.

RATE OTHER ASSETS

User can rate other assets
 -- After submitting the condition form of the nearest assets, the user can rate other assets as well if they wish to.
-- To rate, click on the asset point
-- The condition form will pop-up
